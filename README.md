## Desk Reservation
React + .NET Core website for Desk Reservations.

### Features
* Viewing a list of desks
* Reserving a desk
* Viewing a list of reservations
* Cancelling reservations

### Prerequisites 
* Node.js
* .Net SDK 9

### How to run it
Start the backend
```
dotnet restore
dotnet run
```
Start the frontend
```
npm install
npm run dev
```
